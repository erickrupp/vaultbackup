# **Struktur eines MS**
## **2022-03-07 10:03**

Ein klassisches MS folgt dem [[PDCA-Zyklus]]. Dieser Zyklus dient dazu unterschiedliche Blickwinkel in einer Organisation einzunehmen mit unterschiedlichem Horizont und unterschiedlichem Abstraktionsgrad.

Es soll somit ermöglicht werden, die langen Linien einer strategischen Ausrichtung der Organisation zu beachten um einen groben Rahmen zu setzen innerhalb dessen eine Orientierung möglich ist.

Hierzu werden in allen relevanten MS die einzelnen Anforderungen kapitelweise gleichartig zusammengefasst.

Kap 1-3 sind immer: Anwendungsbereiche, Verweise auf andere Normen und Definition von Begriffen.
