# Note Section
Was ist das Triadische System
?
Das Triadische System ist ein überkommenes System zur Klassifizierung von psychischen Störungen aufgrund ihrer Ursache.
Es wird nicht mehr genutzt, hat aber noch einige Anhänger.
Es unterteilt die Störungen in 3 Klassen:
- psychogen - in der Persönlichkeitsstruktur fußend, oder als Reaktion auf Belastungen
- exogen - in körperlichen Störungen begründet (inkl. Gehirn)
- endogen - keinem der beiden o.g. zuzuordnen
