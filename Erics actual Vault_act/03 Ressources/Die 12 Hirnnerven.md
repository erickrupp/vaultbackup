# **Die 12 Hirnnerven**
## **2022-06-19 18:20**
___
### Die 12 Hirnnerven
#### Allgemeines

Element des [[PNS]]:

| Nerv | Bez                                                      | Was                                                                                                                                                           | s,m,b |
| ---- | -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----- |
| I    | Nervus olfactorus - Riechnerv                            | Nase => Gehirn                                                                                                                                                | s     |
| II   | Nervus opticus - Sehnerv                                 | Netzhaut => Gehirn                                                                                                                                            | s     |
| III  | Nervus oculomotorius - Augenmotorik                      | Steuerung Augenbewegungen                                                                                                                                     | m     |
| IV   | Nervus trochlearis - Augenrollnerv                       | Steuerung schräger oberer Augenmuskel                                                                                                                         | m     |
| V    | Nervus trigminus - Drillingsnerv                         | Augennerv (Nervus ophthalmicus), Oberkiefernerv (Nervus maxillaris), Unterkiefernerv (Nervus mandibularis); sensibel Gesichtsbereich; motorisch Kaumuskulatur | b     |
| VI   | Nervus abducens - Augenabziehnerv                        | lateraler (seitlicher) Augenmuskel                                                                                                                            | m     |
| VII  | Nervus facialis - Gesichtsnerv                           | Steuert Gesichtsmuskeln, innerviert alle Kopfdrüsen (außer Ohrspeicheldrüse); Geschmackswahrn. aus vorderen 2/3 der Zunge                                     | b     |
| VIII | Nervus vestibulocochlearis - Hör- und Gleichgewichtsnerv | Gehör und Gleichgewichtsorgan => Gehirn                                                                                                                       | s     |
| IX   | Nervus glossopharyngeus - Zungen-Rachen-Nerv             | Geschmack hinterer Zungenabschnitt; innerviert Muskeln Rachen (schlucken); innerviert Ohrspeicheldrüse                                                        | b     |
| X    | Nervus vagus - 'umherschweifender' Nerv                  | Hauptnerv des Parasympathikus; Reguliert die Tätigkeit vieler innerer Organe                                                                                  | b     |
| XI   | Nervus accessorius - Beinnerv                            | Steuerung der Beinmuskulatur (Musculus trapezius und sternocleidomastoideus); eigentlich Rückenmark, geht aber auch in die Schädenhöhle                       | m     |
| XII  | Nervus hypoglossus - Unterzungennerv                     | Steuerung der Zungenbewegung                                                                                                                                  | m      |

#### Beispiele für Merksätze:

Nach Namen:
Oh, oh, oh, tausender Tannen arbeiten für viel Geld, vor allem (als) Hebammen.
Oma oben ohne tanzt tropfnass aber froh vor Großvater Alberts Haus

Nach Funktion (sensibel, motorisch, beides)
Some students make money but my brother says Boris Becker makes more.

#### Diagnostische Überprüfungsmöglichkeiten an den Hirnnerven

Neurologische Symptome können sein:
- Schwindel
- Schmerzen (etwa unspezifische Kopfschmerzen)
- Sehstörungen
- Gleichgewichtsstörungen
- Missempfindungen (Parästhesien)
- Kräfteverlust, Kraftlosigkeit

Unspezifische Symptome können mehreren Krankheitsbildern zugeordnet werden. 
=> Auch wenn psychogener Ursprung vermutet wird, sollten neurologische Ursachen durch entsprechende Diagnose ausgeschlossen werden.

Überprüfungsmöglichkeiten der 12 Hirnnerven

| Nerv | Überprüfungsmöglichkeiten                                                                                                                                        |
| ---- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| I    | Geruchsproben                                                                                                                                                    |

| III  | Prüfung der Augenbewegung (Folgen des Fingers ohne Kopfbewegung)                                                                                                 |
| IV   | siehe III                                                                                                                                                        |
| V    | Drücken der Trigeminusreizpunkte (oder [[Trigeminusneuralgie]])                                                                                                  |
| VI   | siehe III                                                                                                                                                        |
| VII  | Stirn runzeln, Augen zukneifen, Wangen aufblasen                                                                                                                 |
| VIII | Flüstern, Stimmgabeltests nach Rinne oder Weber (Rinne: auf Schläfenbein (Hall und Nachhall), Weber: auf Scheitel (beide Seiten gleichlaut?))                    |
| IX   | Gaumensegel hängt auf der betroffenen Seite herab                                                                                                                |
| X    | Prüfung: steht Zäpfchen schief (Schiefstand zur gesunden Seite hin); Auslösen Würgereflex (ausbleiben nicht gut)                                                 |
| XI   | Schultern gegen Widerstand hochziehen, Kopf gegen Widerstand drehen                                                                                              |
| XII  | Zunge rausstrecken, hängt zur kranken Seite hin ab                                                                                                                                                                 |
