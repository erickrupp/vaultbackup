# **Sender-Empfänger-Modell der Kommunikation**
## **2022-11-08 18:32**
___
Definiert als ein wechselseitiger Prozess, in dem ein Sender einen Inhalt (die Nachricht) sendet und ein Empfänger darüber Feedback gibt.

Das ursprüngliche Modell ist extrem einfach und dann vollendet, wenn eine Nachricht gesendet wird. Im Verlauf der Forschung wurde das Modell dann aber immer weiter auf einen Regelkreis umgedeutet (siehe Abb), so dass der Empfänger, nach empfang der Nachricht seinerseits eine Informationsverarbeitung vornimmt und seinerseits zum Sender einer neuen Nachricht werden kann.

![[Bildschirmfoto 2022-11-08 um 18.39.01.png]]

Das Modell ist vglw. einfach gestrickt und beachtet verschiedene beobachtbare Elemente von Kommunikation nicht[^1] und lässt somit Raum für Interpretation und Missverständnisse.
