# **Gewohnheitskopplung**
## **2022-02-27 16:31**

Durch die Kopplung von Gewohnheiten können Sie eine neue Gewohnheit mit einer aktuellen Gewohnheit verknüpfen. 

Die Formel für die Gewohnheitskopplung lautet: Nach (aktuelle Gewohnheit) werde ich (neue Gewohnheit).
