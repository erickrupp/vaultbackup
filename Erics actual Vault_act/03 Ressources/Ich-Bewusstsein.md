# **Ich-Bewusstsein**
## **2022-06-08 14:27**
___
Das Ich-Bewusstsein ist ein integrierender Begriff, in dem eine Reihe unterschiedlicher Auffassungen und Theorien begrifflich zusammengefasst wird.
In der Regel wird vom [[Ich-Bewusstsein]] in einer Weise gesprochen, dass es sich um das: 
- Selbstbewusstsein (im Sinne dessen, dass eine Person sich bewusst ist eine (und in der Folge auch dieselbe) Person zu sein)
- Ego-Bewusstsein im Sinne der kognitiven Konstruktion der eigenen Individualität, die für die eigene Persönlichkeit gehalten wird. Diese umfasst je nach Theorie:
	- Gedanken
	- Werte
	- [[Glaubenssätze]] 
	- Gefühle
	- typische Reaktionen
	- präferrierte [[Entscheidungsheuristik]]
