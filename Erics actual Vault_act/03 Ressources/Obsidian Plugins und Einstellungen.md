# Einstellungen und Plugins für mein Vault

## Einstellungen:
- Editor
	- Live Editor anzeigen
	- Frontmatter anzeigen
	- Zeilennummern anzeigen
	- Rechtschreibprüfung an
- Dateien und Links
	- Links aktualisieren Ein
	- Standardordner für Anhänge: Attachments
	- Ignorierte Dateien: Ordner Attachments und Templates
	- Darstellung: Themes Standard Dunkel oder Primary Hell oder Dunkel
## Obsidian Erweiterungen:
- Mit Stern
- Templates - Ctrl-Opt-Com-T
## Externe Erweiterungen:
- Advanced Tables
- Calendar
- Dataview
- Footnote Shortcut [[Obsidian Footnotes]]
- Homepage: Landingpage = [[000_Home_Dashboard|HomeNote]]; View = Live Editor View; Dataview Refresh = ON; Open Mode = Keep open Notes
- Kanban (Default Options)
- Kindle Highlights: Default Options; [[Kindle-Sync-Template]]
- Periodic Notes (Not well used at the moment)
- Quick Switcher++: Default Options; Opt-Com-O to open Pane; @ for Symbol list (tags, headers etc in current note); # for Heading search across vault
- Sliding Panes (Andys Mode)
- [[spaced repitition plugin]] (Lernkarten)
- Table of Contents: Opt-I (create TOC); Shift-Opt-I (create TOC for next heading level)
- Tag Wrangler
- Tidy Footnotes [[Obsidian Footnotes]]
