# **Affektbilanz**
## **2022-07-13 15:00**
___
Im Rahmen einer Affektbilanz werden bestimmte Fragestellungen auf 2 Skalen (positiv / negativ) getrennt voneinander bewertet. 

Vorteil ist, dass man die jeweils positiven und negativen Bewertungen nicht auf einer Skala 'verrechnen' muss, sondern getrennt voneinander betrachten kann. So werden  Positionen der einen Seite nicht von Positionen der anderen Seite von vorneherein beeinflusst.

Anwendungen im Rahmen von Trainings, wenn es darum geht, wie man sich mit etwas fühlt.

Anwendung auch im Rahmen des [[ZRM]]. Siehe hier bspw. die [[Machen Sie doch, was Sie wollen! - Storch, Maja#^fc7dc5|Wurmbilanz]] von Maja Storch.
