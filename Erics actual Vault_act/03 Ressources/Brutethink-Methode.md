# **Brutethink-Methode**
## **2022-11-07 12:16**
___
Methode ähnlich der [[Reizwortanalyse]]. Es geht um Inspiration mit willkürlichem, nichts mit dem eigentlichen Thema zu tun habendem Input.

- Eine Problemstellung oder eine Zielsetzung muss eindeutig beschrieben werden
- Es wird z.B. ein Begriff oder ein Bild willkürlich aus einer völlig unabhängigen Quelle (bspw. Lexikon) herausgesucht.
- Dieser Begriff oder dieses Bild wird dann in Beziehung gesetzt zur ursprünglichen Problemstellung.
- Ideen werden gesammelt 
- Ideen werden anschließend geclustert und bewertet bspw. mit:

```dataview
LIST
FROM #businesstraining/methoden/bewertung 
```
