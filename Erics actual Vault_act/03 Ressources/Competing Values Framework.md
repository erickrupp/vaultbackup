# **Competing Values Framework**
## **2022-05-23 11:07**
___
Das Competing Values Framework wurde 1983 von Robert Quinn und John Rorbaugh entwickelt. Es basiert auf der Annahme, dass sich Werte identifizieren lassen, die alle für den Unternehmenserfolg relevant sind, die sich jedoch auf unterschiedlichen Extremen von 2 Achsen darstellen lasse, somit also im Wettbewerb zueinander stehen.

Im CVF werden die beiden Achsen:
- Stabilität - Flexibilität
- interner Fokus - externer Fokus

unterschieden und als Koordinatensystem dargestellt. Somit ergeben sich 4 Quadranten und innerhalb dieser je 2 Werte oder Rollen, je nachdem welcher Achse sie mehr zuneigen.

Abbildung:

![[Rollen einer Führungskraft nach Robert Quinn.png]]

Die 4 Quadranten können wie folgt überschrieben werden:
1. Intern-Stabil
   Hierarchische Kultur
   Command and Control
   Werte: Effizienz, Disziplin, Konsistenz, Uniformität
   Leadertyp: Koordinator, Kontrolleur
2. Intern-flexibel
   Clan-/Familienkultur
   Zusammenarbeit
   Werte: Kommunikation, Entwicklung, Verpflichtung
   Leader: Mentor, Facilitator, Teambuilder
3. extern-flexibel
   Adhocracy-Kultur (Inkrementalismus)
   Create, Productivity
   Werte: Transformation, Agilität, Innovation
   Leadertyp: Innovator, Entrepreneur, Visionär
4. extern - stabil
   Marktgetriebene Kultur
   Wettbewerb
   Werte: Zahlen, Zielerreichung, Leistungssteigerungen
   Leadertyp: Competitor, Regisseur

Welche Elemente für welches Unternehmen oder für welchen Unternehmensteil relevant sind, muss in der [[Unternehmensstrategie]] festgelegt werden. Es ist notwendig, die entsprechenden Anforderungen auch in der [[MOC Unternehmensführung]], der Aufbauorganisation und im [[Person-Job-Fit]] abgebildet zu haben.

Hinweis: 
Die stabilitätsgetriebenen Quadranten kommen im CVF nicht so richtig gut weg. Aber auch sie können einen guten Wert in Unternehmen besitzen, wenn das Unternehmen sein [[Golden Circle - Sinek, Simon|WARUM]] kennt und verstanden hat, dass Business ein [[offene und geschlossene Spiele|offenes Spiel]] ist.
