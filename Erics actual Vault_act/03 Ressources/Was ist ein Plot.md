# **Was ist ein Plot**
## **2022-11-10 16:55**
___
### Allgemeines
Der Autor von [[20 Masterplots - Ronald Tobias]] legt wert darauf, dass ein Plot nicht einfach nur eine Struktur, ein Skelett einer Geschichte ist.

Er ist eher eine Kraft, die alle Elemente der Geschichte sinnvoll zusammenhält.

Eine Geschichte besteht aus chronologischer Abfolge von Dingen (zuerst das, dann das, dann das usw. usf.)

Ein Plot verbindet die einzelnen Elemente einer Geschichte. "Der König stirbt, die Königin stirbt" ist eine Geschichte, "Der König stirbt und dann stirbt die Königin aus Kummer", ist bereits ein einfacher Plot.

Der komplexere Plot benötigt in der Regel Ursache und Wirkung, Absicht und Motiv. Warum machen die Figuren das und das.

Die Frage nach der Absicht, der Motivation der Figuren in der Geschichte führt somit zum Plot hin. 

Der gute Plot benötigt jedoch eine Auflösung aller Fragen, damit die Geschichte ein befriedigendes Ende finden kann.

Nach Aristoteles verläuft der klassische Plot auf dem Weg zu diesem Ende über 3 Abschnitte

Anfang, Mitte, Ende 

### Aufgaben der Teile für den Plot
#### Anfang 
- Präsentation des Problems das gelöst werden muss
- Definiert die Wünsche der Hauptpersonen (entweder Glück oder Unglück) Fragen hierzu: Was wünscht sich meine Figur, was braucht sie, was treibt sie an?
- Legt so die Motivation dafür fest, dass die Figur in der Folge handelt, wie sie handelt.

#### Mitte 
- Die Figur verfolgt ihr Ziel, geht ihren Beweggründen nach, strebt ihren Wünschen zu.
- Die Figur stößt auf Probleme, die sie an der Ausführung ihrer Handlungen hindern (*Wendungen*, bzw. Plot-Points)
- Durch die Abarbeitung an den Problemen, und deren (hoffentlich erfolgreichem Ende) folgt das *Wiedererkennen*. Das bedeutet die Veränderung der Beziehung der Figuren nach durchlaufen der Wendung.
- Oft werden im Film 2 Plot-Points eingebaut 

#### Ende
- Lösung. Das logische Ergebnis der ersten beiden Abschnitte 
- alles wird aufgedeckt und beantwortet
- keine relevante Frage bleibt offen
