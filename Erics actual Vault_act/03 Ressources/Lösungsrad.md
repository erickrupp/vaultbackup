# Lösungsrad
## 2023-02-04 16:15
___
### Allgemeines
Methode zum systemischen Erfragen verschiedener Aspekte der Lösungsfindung.


### Durchführung

1. Beschreiben des Problems
   - Was ist konkret passiert?
   - Wo ist das Problem aufgetreten?
   - Wer? Was? Wann? Wo? Wie?

2. Beschreiben des Zielzustands
   - Was soll anstelle des Problems treten?
   - Was ist das Ziel?
   - Wie wird es sein, wenn das Problem gelöst ist?
	[[Ideales Interview nach Carmen Kindl]] hier Vision und was genau soll anders werden

3. Ausnahmen vom Problem 
   - In welchen Situationen ist das Problem nicht aufgetreten?
   - Was ist dann anders?
   - Was machen Sie / wer anderes anders 

4. Lösungsversuche und Erfahrungen
   - Was haben Sie schon versucht?
   - Welche Erfahrungen wurden gemacht? Was hat geholfen, was nicht?
   - Welche Ressourcen wurden entdeckt?
     [[Ideales Interview nach Carmen Kindl#^272ad8]]
	[[NLP HELP]] hier *p*acen
	
5. Zuversicht, das Problem zu lösen 
   - Was spricht für eine Lösung?
   - Was ermutigt sie?
   - [[Skalierungsfragen]] Wie zuversichtlich sind sie auf einer Skala von 1-10, dass es klappt?

6. Beteiligte Ressourcen
	- Wer ist beteiligt?
	- Wer könnte zur Lösung beitragen?
	
7.  Meilensteine in Richtung Lösung 
	- [[Wunderfragen]] Nehmen wir an, das Problem ist gelöst, wie haben sie das geschafft?
	- [[Skalierungsfragen]] Auf einer Skala von 1-10, wie weit sind Sie mit der Lösung schon gekommen?
	- Woran erkennen Sie, dass Sie einen Schritt weitergekommen sind. 

*Weitere Hinweise*
Kann, wie oben gezeigt mit einigen anderen Tools kombiniert werden oder zahlt auf ähnliche Zwecke ein.
