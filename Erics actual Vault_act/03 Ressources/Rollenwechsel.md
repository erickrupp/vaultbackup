# Rollenwechsel
## 2022-12-18 16:32
___
### Allgemeines
Methode zur Bearbeitung eines Themas, einer Fragestellung aus der Sicht einer anderen Person (eines Ratgebers)
Ein [[Gedanken- Fokussprint|Fokussprint]] aus der Perspektive einer anderen Person

### Durchführung

1. Schreibe das Thema als Überschrift auf 
2. Überlege: Von wem kann ich in dieser Sache etwas lernen? Wer ist in dieser Frage ein Vorbild?
3. Schreibe aus der Position dieser Person einen kurzen ratgebenden Text an dich selbst.
4. Am Ende markiere das wichtigste und halte es in einem Kernsatz fest.
