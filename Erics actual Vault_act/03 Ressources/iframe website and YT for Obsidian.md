# **iframe embed for Obsidian**
## **2022-07-19 11:44**
___
Um eine Webseite einzubinden nutze iframe mit folgendem code:

```html
<iframe
		height = 800
		width = 100%
		src="https://INSERT_WEBSITE/"></iframe>
```

For YT go to YT
- Start the video you want to embed
- klick on share (Teilen)
  ![[Bildschirmfoto 2022-07-19 um 11.56.52.png]]
In the following menu klick on embed (einbetten)
![[Bildschirmfoto 2022-07-19 um 11.58.11.png]]

Then decide whether you want to start at the postition you are currently or from the beginning and klick copy (Kopieren)
![[Bildschirmfoto 2022-07-19 um 11.59.13.png]]

Paste the code into a note in Obisidan like this (first the pasted Code as manual created code block thereafter the code without using backticks so that the video can be watched):
```html 
<iframe width="560" height="315" src="https://www.youtube.com/embed/EZJguXF_6m8" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
```
