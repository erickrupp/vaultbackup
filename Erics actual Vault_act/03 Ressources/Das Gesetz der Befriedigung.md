# **Das Gesetz der Befriedigung**
## **2022-02-27 17:52**

Das vierte Gesetz der Verhaltensänderung lautet: Die Gewohnheit muss befriedigend sein.  
  
Wir wiederholen ein Verhalten bereitwilliger, wenn es befriedigend ist.  
  
Das menschliche Gehirn hat sich so entwickelt, dass es unmittelbaren Belohnungen Vorrang vor zeitverzögerten gibt.  
  
Das oberste Gebot der Verhaltensänderung lautet: 

>**Was sofort belohnt wird, wird wiederholt. Was sofort bestraft wird, wird vermieden. 
  
Damit eine Gewohnheit Bestand hat, ist ein **unmittelbares Erfolgserlebnis** nötig – auch wenn es nur in einer Kleinigkeit besteht.  
  
Die ersten drei Gesetze der Verhaltensänderung – Die Gewohnheit muss offensichtlich sein, sie muss attraktiv sein und sie muss einfach sein – erhöhen die Wahrscheinlichkeit, dass ein Verhalten diesmal ausgeführt wird. Das vierte Gesetz der Verhaltensänderung – Sie muss befriedigend sein – erhöht die Wahrscheinlichkeit, dass es beim nächsten Mal wiederholt wird.
