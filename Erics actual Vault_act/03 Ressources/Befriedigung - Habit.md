# **Befriedigung - Habit**
## **2022-02-27 17:54**

Das Gefühl, Fortschritte zu machen, ist sehr befriedigend.  
  
Mit einem **Gewohnheitstracker** lässt sich ganz einfach festhalten, ob Sie eine Gewohnheit ausgeführt haben – zum Beispiel, indem Sie einen Tag im Kalender ankreuzen.  
  
Gewohnheitstracker und andere sichtbare Messgrößen können Ihre Gewohnheiten befriedigend gestalten, weil sie klare Beweise für Ihren Fortschritt liefern.  
  
Lassen Sie eine Serie nicht abreißen. Versuchen Sie, Ihre Gewohnheitsserie fortzusetzen. Setzen Sie nie zweimal hintereinander aus. Wenn Sie einmal aussetzen mussten, sollten Sie so schnell wie möglich wieder auf Kurs kommen. Nur weil man etwas messen kann, ist es noch lange nicht das Wichtigste.
