# **Denkpfad**
## **2022-12-14 11:33**
___
### Allgemeines
Methode zur ==Mikro-Strukturierung== eines Textes.
Strukturierung der eigenen Gedanke im Rahmen von [[Selbstmanagement]] möglich.
Stark assoziativ

### Durchführung

1. Augen schließen
2. fokussieren auf das bevorstehende Schreibthema
3. Versuch erste Wegweiser in der Vorstellung zu erkennen (Stichworte, Überschriften, Gliederungspunkte)
4. Notieren dieser einzelnen Strukturen auf ein Blatt Papier - wie einen Parcours, den sie gleich gedanklich durchlaufen werden (gerne auch mit Skizzen). 
5. Visualisierung dort aufhängen, wo sie im Schreibprozess gut sichtbar ist, damit Sie sich immer mal wieder daran kalibrieren können.

Variation:
In Step 4:
- Aufschreiben der Gedanken auf Kartei- oder Mod-Karten.
- Als einen separaten Punkt diese Karten als [[Gedankenlandkarte]] oder [[Mindmapping|Mindmap]] zurechtlegen lassen.

Kann gut als Vorbereitung zur Methode [[Textpfad]] genutzt werden.
