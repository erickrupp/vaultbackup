# **Das Gesetz der Attraktivität**
## **2022-02-27 17:30**

Das zweite Gesetz der Verhaltensänderung lautet: Die Gewohnheit muss attraktiv sein.  
  
Je attraktiver etwas ist, desto wahrscheinlicher wird es zur Gewohnheit.  
  
Gewohnheiten sind eine **dopamingesteuerte Feedbackschleife**. 

Steigt der Dopaminspiegel, so steigt auch unsere Motivation zu handeln. Die **Erwartung einer Belohnung veranlasst uns zum Handeln**, nicht die Belohnung selbst. Je größer die Erwartung, desto größer die Dopaminspitze.  
  
Mit Bedürfniskombinationen können Sie Ihre Gewohnheiten attraktiver machen. Dazu wird eine Aktion, die Sie durchführen möchten, mit einer Aktion kombiniert, die Sie ausführen müssen.
