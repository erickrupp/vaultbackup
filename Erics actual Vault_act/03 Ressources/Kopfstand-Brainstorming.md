# **Kopfstand-Brainstorming**
## **2022-05-22 11:22**
___
Beim Kopfstand-Brainstorming handelt es sich methodisch um ein einfaches [[Brainstorming]] oder [[Brainwriting]]. 

Das Kopfstand-Brainstorming zeichnet sich dadurch aus, dass die Fragestellung (in der [[Kollegiale Beratung]] die Schlüsselfrage) auf den 'Kopf gestellt' wird. Sie entspricht so den Paradoxen Fragen aus dem Pool der [[Systemische Fragen|Systemischen Fragen]].
