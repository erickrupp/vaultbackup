# Supervisionen
## **2022-09-28 15:39**
___
Begleitung von Mitarbeitenden durch Erfahrene Dritte, bspw. innerhalb von Coachings, bei Üben von neuen Fähig- und Fertigkeiten, in Situationen in denen eine Fremdanalyse vorteilhaft sein kann, etc.

Der Supervisor beobachtet wertschätzend und gibt [[Wichtigkeit von Feedback|Feedback]]. 

Es wird so ermöglicht die eigenen Muster und Blinden Flecke zu beleuchten und die persönliche Entwicklung zu bereichern.

Supervision kann Teil einer [[Feedbackkultur]] sowie einer [[Positive Fehlerkultur]] in einer Organisation sein.

Wie bei Feedback wird das Aushalten von Rückmeldungen und die fruchtbare Integration dieser Rückmeldungen in das eigene Verhalten eingeübt.
